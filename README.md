# flight-simulator

Proyecto para simular vuelos.

Tecnologías:

- Backend: express
- Base de datos: mongodb

## Documentación

## End-point: Create Flight

### Method: POST

> ```
> /flights
> ```

### Body (**raw**)

```json
{
  "flightCode": 123
}
```
